Protótipo de Sistema para o Aeroclub Asas Abertas,feito no Rise UP 2026.1 do Porto Digital com a empresa NTTData:
A Plataforma web centralizada para o Aeroclube Asas Abertas, integrando cadastro, agendamento de aulas, diário do aluno e professor, controle financeiro e documentos, eliminando a gestão manual.
A solução entregue é uma plataforma web front-end, desenvolvida em HTML, CSS3 e JavaScript, sem banco de dados, focada nas funcionalidades principais solicitadas pela empresa NTTData para o Aeroclube Asas Abertas de Pernambuco. O sistema centraliza o cadastro de alunos e professores, o agendamento de aulas e serviços, e o registro e visualização do histórico de aulas por perfil de usuário.

Site: https://asasabertas-aero-clube.vercel.app/
