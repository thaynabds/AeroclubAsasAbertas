# Sistema de Gerenciamento de Agenda e Aulas — Aeroclube Asas Abertas 🛩️

**Protótipo desenvolvido no RiseUp 2026.1 | Porto Digital + NTT DATA via Residência em Software & IA** 

Plataforma web para digitalizar a gestão administrativa do **Aeroclube Asas Abertas** de Pernambuco, eliminando o uso de planilhas e papéis que geravam conflitos de horários, erros financeiros e riscos operacionais.

---

## 🎯 Sobre o Projeto

O Aeroclube Asas Abertas precisava de uma solução moderna para centralizar sua gestão. Desenvolvemos um **protótipo funcional** de sistema web que atende às principais necessidades do aeroclube:

- Cadastro centralizado de alunos e instrutores
- Agendamento digital de aulas e serviços
- Diário de bordo digital (Professor e Aluno)
- Controle de acesso por perfil (Admin, Instrutor, Aluno)
- Aplicação automática de regras da aviação civil (ANAC)
- Histórico completo de aulas e pagamentos

### Tecnologias Utilizadas

- **Frontend**: HTML5, CSS3 e JavaScript (Vanilla)
- **Prototypagem**: Figma
- **Deploy**: Vercel
- **Metodologia**: Scrum (RiseUp 2026.1)

---

## ✨ Funcionalidades Principais

- ✅ **Login por perfil** (Administrador, Instrutor e Aluno)
- ✅ **Cadastro de Alunos e Instrutores**
- ✅ **Agendamento de aulas** com verificação de disponibilidade
- ✅ **Diário do Professor** e **Diário do Aluno**
- ✅ **Dashboard** personalizado por tipo de usuário
- ✅ Design responsivo e interface intuitiva

## Como Executar Localmente

1. Clone o repositório:

   ```bash

   git clone https://github.com/thaynabds/AeroclubAsasAbertas.git

   ```

2. Acesse a pasta do projeto:

   ```bash

   cd AeroclubAsasAbertas

   ```

3. Abra o arquivo `index.html` no navegador.

*(Não requer instalação de dependências ou backend — projeto 100% frontend)*
